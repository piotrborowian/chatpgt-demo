----- Product description -----
I want to build a clone of the ChatGPT web client.
The website should have a modern slick user interface where the user can chat with the GPT-4o model from Open AI.
The website should support conversation history, where the user can create a new conversation, or choose to return
to the previous conversation. 
A conversation, of course, should preserve its context accross the duration of the conversation.
Use OpenAI API and Supabase (project chat-gpt-demo) for storage in your implementation

I prepared a .env.local file with the following keys:

OPENAI_API_KEY 
NEXT_PUBLIC_SUPABASE_URL
NEXT_PUBLIC_SUPABASE_ANON_KEY

The .cursor/mcp.json contains a supabase MCP server definition that you should use to access supabase to create tables etc.
Install the server in any way you need, if it's not already installed.
